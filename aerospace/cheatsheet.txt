 AeroSpace Keybindings

 WORKSPACES
   alt + <key>                   switch to workspace
   alt + shift + <key>           move window to workspace
   alt + tab                        back-and-forth

   main monitor:
   A aws console          P postman              S slack
   T terminal (Ghostty)   W work browser

   built-in monitor:
   C personal browser   M meetings           O other
   V vpn                Z zoom

 FOCUS / MOVE / JOIN
   alt + h j k l                  focus left/down/up/right
   alt + shift + h j k l          move  left/down/up/right
   alt + cmd + h j k l            join  left/down/up/right

 LAYOUT
   alt + cmd + f                    fullscreen
   alt + ,                          layout accordion horizontal vertical
   alt + /                          layout tiles horizontal vertical
   alt + =                          resize smart +50
   alt + -                          resize smart -50

 MONITORS
   alt + ctrl + tab                 move-workspace-to-monitor next
   ctrl + cmd + [                   focus-monitor prev
   ctrl + cmd + ]                   focus-monitor next
   ctrl + cmd + shift + [           move-node-to-monitor prev
   ctrl + cmd + shift + ]           move-node-to-monitor next

 OTHER
   alt + cmd + =                    balance-sizes
   alt + cmd + s                    screencapture -i -c
   shift + cmd + h                  swap left
   shift + cmd + j                  swap down
   shift + cmd + k                  swap up
   shift + cmd + l                  swap right

 SERVICE MODE                    alt + ctrl + ;
   del                              close-all-windows-but-current
   down                             volume down
   esc                              reload-config
   f                                layout floating tiling
   r                                flatten-workspace-tree
   shift + down                     volume set 0
   up                               volume up

 DISMISS                         esc  or  q
